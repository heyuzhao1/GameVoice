# GameVoice 用户指南

## 🎮 软件概述

GameVoice 是一款专为游戏玩家设计的低延迟语音通话程序，特别优化同城玩家间的连接。它具有简洁易用的界面、极低的资源占用，以及优秀的语音质量。

## ✨ 核心特性

### 1. **极低延迟**
- 同城玩家间延迟 < 50ms
- 跨城玩家间延迟 < 100ms
- 智能路由优化

### 2. **高性能**
- CPU 占用 < 5%（空闲时）
- 内存占用 < 100MB
- 不影响游戏性能

### 3. **优质音频**
- 智能降噪和回声消除
- 语音激活检测
- 自适应音频编码

### 4. **简单易用**
- 直观的游戏风格界面
- 一键加入房间
- 实时状态显示

## 🖥️ 界面功能详解

### 顶部状态栏
- **程序名称和状态**：显示连接状态（已连接/未连接）
- **网络指标**：实时显示延迟和带宽
- **设置按钮**：快速访问设置面板

### 左侧控制面板
#### 房间控制
- **房间ID输入**：输入或创建房间
- **加入/离开按钮**：一键操作
- **用户计数**：显示当前在线人数

#### 麦克风控制
- **静音开关**：一键静音/取消静音
- **音量调节**：滑动条控制麦克风音量
- **设备选择**：选择音频输入设备

#### 连接统计
- **延迟**：实时语音延迟（ms）
- **丢包率**：网络质量指标（%）
- **带宽**：当前使用的带宽（kbps）

### 右侧用户列表
- **在线用户**：显示所有房间成员
- **说话状态**：绿色指示灯表示正在说话
- **音量条**：显示每个用户的音量大小
- **用户信息**：显示用户ID和状态

### 底部状态栏
- **性能监控**：显示CPU和内存占用
- **版本信息**：当前软件版本
- **版权信息**

### 设置面板
- **音频设备**：选择输入/输出设备
- **音频质量**：选择音质预设
- **网络优化**：启用低延迟模式等
- **快捷键**：自定义操作快捷键

## 🚀 快速开始

### 第一步：启动程序
```bash
cd GameVoice
npm run dev
```
然后在浏览器中访问：http://localhost:5173

### 第二步：创建或加入房间
1. 在左侧面板输入房间ID
2. 点击"加入房间"按钮
3. 等待连接建立

### 第三步：配置音频
1. 确保麦克风已连接
2. 调整合适的音量
3. 测试语音激活

### 第四步：开始通话
1. 查看用户列表确认其他玩家在线
2. 开始正常说话
3. 监控连接质量

## 🔧 故障排除

### 常见问题

#### 1. **无法听到声音**
- 检查扬声器/耳机连接
- 确认音量未静音
- 检查系统音频设置

#### 2. **麦克风不工作**
- 检查麦克风权限
- 确认选择了正确的输入设备
- 测试麦克风在其他应用中的表现

#### 3. **高延迟问题**
- 关闭其他占用带宽的应用
- 使用有线网络连接
- 选择最近的服务器区域

#### 4. **连接不稳定**
- 检查网络连接
- 重启路由器和调制解调器
- 尝试使用不同的网络

### 调试模式
启动时添加调试参数：
```bash
npm start -- --debug
```

查看日志文件：
- Windows: `%APPDATA%/GameVoice/logs/`
- macOS: `~/Library/Logs/GameVoice/`
- Linux: `~/.config/GameVoice/logs/`
